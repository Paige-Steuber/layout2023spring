Game Concept

Paige's Collection Game (title pending)

My game is a collection type game, inspired by indie open world life simulation video games, like "Slime Rancher" mixed with dungeon crawler aspects, like "The Cult of the Lamb." The goal of the game would to be to build your "farm" or collection of things. In the case of my game you would be trying to fill a museum with items such as rare flowers, bones, or treasures. The home base, or museum, might be filled with different tasks that run in the background that you have to check on when you're not out exploring or dungeon crawling. Different objectives would include:

  • Keeping up on these tasks, for example picking flowers that you grow.
  • Survive in the dungeon.
  • Explore the unknown regions around you.

Ui Elements

UI elements include:

  • Collection slots
  • Inventory
  • Tasks / goal tracker
  • Health and energy bars
  • A clock of some type
